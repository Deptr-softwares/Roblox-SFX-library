# JJS-Skill-Builder-SFX-Super-Library
This is a library/resources page for skill builders or Roblox development in general. You sort through this library using tags or search here, You're welcome

There will be audio names, audio names on roblox creator store, their ids, their duration.

You can take an audio id put them into each "block" in the audio preview page, and listen to them there, where you can speed up/slow down it, set the starting position and the end position, and how loud it is. The default settings follow JJS (Jujutsu Shenanigans) skill builder's settings, which is: "Volume: 0.5", "Start: 0", "End: 500", "Speed: 1".

Also everything is free, you can copy these amateurish codes, not like you need them anyway right?
